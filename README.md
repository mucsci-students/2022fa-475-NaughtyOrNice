## Controls
W - Move Forward <br/>
S - Move Backward <br/>
A - Strafe Left <br/>
D - Strafe Right <br/>
Space - Jump <br/>
Left Click - Shoot Coal <br/>
Right Click - Shoot Presents <br/>
Esc - Pause
